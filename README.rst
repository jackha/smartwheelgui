Prerequisities
--------------

python 3.4

pyserial::

    $ pip install pyserial

macosx::

new ActiveTcl

https://www.python.org/download/mac/tcltk/


Features
--------


Start the gui::

    $ python3 gui.py

Start the config screen::

    $ python3 config.py


In the gui, you can:

- Connect 

- Reset

- Type in commands manually

- Enable, disable

- Each wheel has a connection configuration as wel as a wheel configuration
